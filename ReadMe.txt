В папке Test_task лежит приложения для сервера

В папке client лежит приложение для клиента

На вход программы принимаю 2 значения сначало IP потом Порт

Для поиска файла нужно указать полный путь к файлу

Клиент(test_client): 
1.	Чтение строк из std::cin. Если в строке не 2 слова (разделитель - пробел), то такая команда считается невалидной, и клиент в ответ пишет сообщение об ошибке. Валидная строка из двух слов отсылается серверу. Окончание команды - символ ‘\n’.
2.	Адрес и порт сервера передаются в командной строке.

Сервер(test_server): 
1.	Получает 2 слова: кодовое слово и имя файла, ищет количество вхождений данного кодового слова в файле и отправляет результат клиенту. Файлы в кодировке UTF-8.
Адрес и порт для сервера передаются в командной строке
